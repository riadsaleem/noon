# موقع شركة خمسة نون العربية

موقع إلكتروني احترافي لشركة خمسة نون العربية للتجارة والتسوق.

## المميزات

- ✅ 4 صفحات رئيسية (الرئيسية، الأقسام، الفروع، لوحة التحكم)
- ✅ 18 قسم مع صورهم
- ✅ 3 فروع مع معلوماتهم وروابط خرائط جوجل
- ✅ نظام تسجيل دخول محمي
- ✅ تصميم متجاوب على جميع الأجهزة

## التقنيات المستخدمة

- React 19 + TypeScript
- Tailwind CSS
- Express.js
- MySQL
- Vite

## التثبيت

```bash
# تثبيت المكتبات
pnpm install

# تشغيل وضع التطوير
pnpm dev

# بناء للإنتاج
pnpm build

# تشغيل الإنتاج
NODE_ENV=production node dist/index.js
```

## متغيرات البيئة

أنشئ ملف `.env` في المجلد الرئيسي:

```
DATABASE_URL=mysql://user:password@host:port/database
JWT_SECRET=your-secret-key
NODE_ENV=production
```
